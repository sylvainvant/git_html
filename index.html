Installer Git

$ sudo apt-get install git


Configuration

Définir le nom que l'on veut pour faire nos Commits

$ git config --global user.name "[name]"

Définir email que l'on veut pour faire nos Commits

$ git config --global user.email "[email address]"

Active la colorisation

$ git config --global color.ui auto


créer des repositories

Crée un nouveau référentiel local avec le nom spécifié

$ git init [project-name]

Télécharge un projet et l'historique complet de ses versions

$ git clone [url]


ligne de commande

ajouter un ou des fichiers

git add [path + nom du fichier]

Ajouter tous les fichiers modifiés :

git add .

commiter un ou des fichiers

git commit -m "votre commentaire"

si vous voulez faire un add en même temps :

git commit -am "votre commentaire"

créer une branche

git checkout -b [ma-branche]

supprimer un branche

git branch -D [ma-branche-a-supprimer]

attention de ne pas être sur cette branche, sinon elle ne sera pas supprimé

Lister les branches en local

git branch

pusher une branche

git push origin [ma-branch]

récuperer le contenu d'une branche sur le remote

git pull origin [ma-branch]


Faire des changements

Répertorie tous les fichiers nouveaux ou modifiés à valider

$ git status

Instantanés du fichier en préparation du contrôle de version

$ git add [file]
